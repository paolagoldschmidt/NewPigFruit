# NewPigFruit

NewPigFruit e um jogo arcade em Phaser onde voce controla um pig para coletar frutas que caem do topo da tela.

## Como jogar
- Mova com as setas esquerda e direita.
- O objetivo de cada level e coletar a quantidade exigida de banana, ma√ßas e limao antes do tempo acabar.
- O item `baccon` funciona como penalidade e zera o placar das frutas.
- Use `ENTER` para avancar entre levels quando indicado.
- Use `R` para reiniciar a partida.

## Estrutura de levels
- **Level 1**: 5 de cada fruta em 60s
- **Level 2**: 10 de cada fruta em 120s
- **Level 3**: 13 de cada fruta em 150s

Ao vencer o Level 3, o jogo abre uma tela final com animacao de danca do pig.

## Executar localmente
```bash
npm install
npm start
```
